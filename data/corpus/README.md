# Corpus — Termos e Políticas de Serviço

Diretório reservado para os documentos-fonte (termos de uso / políticas de privacidade).

## Escopo

| Plataforma | Documento               | Status     |
|-----------|-------------------------|------------|
| X (Twitter) | Termos de Serviço       | pendente   |
| X (Twitter) | Política de Privacidade | pendente   |
| WhatsApp   | Política de Privacidade | pendente   |
| Instagram  | Termos de Uso           | pendente   |
| Facebook   | Termos de Serviço       | pendente   |
| Facebook   | Política de Dados       | pendente   |

## Formato esperado

- Arquivos `.txt` (texto plano) ou `.md`
- Nomeação: `{plataforma}_{tipo}_{data_coleta}.txt`
  - Ex: `x_privacy_2026-01.txt`, `whatsapp_privacy_2026-01.txt`
- Sem PDF/OCR nesta versão do MVP.

## Instruções

1. Acesse a página oficial do termo/política.
2. Copie **todo** o texto da página.
3. Cole em um arquivo `.txt` neste diretório.
4. Registre a data de coleta no nome do arquivo.
