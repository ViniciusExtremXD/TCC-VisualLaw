# Saidas de Referencia

Este diretorio armazena saídas exportaveis do MVP em formato util para evidencias de orientacao e banca.

Arquivos esperados:

- `clauses.json`
- `highlights.json`
- `explanations.json`
- `traceability.json`

Para regenerar os arquivos de referencia, execute:

```bash
npm run sync:research
```
