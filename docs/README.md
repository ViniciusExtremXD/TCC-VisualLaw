# Documentacao Academica

Este diretorio passa a concentrar os materiais de suporte ao TCC.

## Estrutura

- `docs/metodologia`: problema, objetivos, criterios analiticos, escopo e dicionario lexico
- `docs/sdd`: contratos executaveis de escopo, arquitetura, pipeline, dados, validacao e harness
- `docs/diagramas`: figuras e representacoes de arquitetura, atividade e navegacao
- `docs/validacao`: protocolo, questionarios, teste de compreensao e coleta de metricas
- `docs/evidencias`: auditoria produto versus TCC, backlog e roteiro de demonstracao
