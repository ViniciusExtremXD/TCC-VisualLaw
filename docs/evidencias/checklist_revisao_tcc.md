# Checklist de Revisao Critica do MVP

## P0

- consolidar o lexicon academico como fonte unica do motor
- manter home orientada a tradutor de juridiques, nao a demo generica
- garantir comparacao entre texto original e linguagem simples no Reader
- expor impacto, direito LGPD e leitura para validacao por clausula
- manter rastreabilidade exportavel e relatorio academico funcionando

## P1

- reduzir nomenclatura residual de interface "premium/cupertino" no codigo
- configurar lint de forma nao interativa para o projeto
- ampliar FAQ e observacoes metodologicas do lexicon
- consolidar evidencia de validacao com resultados empiricos apos prova de conceito

## P2

- refinar o visual do PDF para uso direto no artigo
- criar visao agregada por documento-base
- incluir metricas locais opcionais de navegacao para estudos futuros
