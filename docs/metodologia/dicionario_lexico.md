# Dicionario Lexico como Cerne da Solucao

## Papel no TCC

O dicionario lexico e o principal artefato de pesquisa aplicada do projeto porque ele conecta:

- o problema de compreensao
- a operacionalizacao do pipeline
- a explicacao em linguagem simples
- a camada visual do Visual Law
- a rastreabilidade entre termo detectado e saida exibida

## Estrutura da entrada

Cada entrada do lexicon contem, no minimo:

- identificacao do termo
- traducao direta
- definicao leiga
- exemplo pratico
- categoria tematica
- icone visual
- nivel de impacto
- direito LGPD relacionado
- palavras-chave relacionadas

## Funcao tecnica

No sistema, o dicionario serve para:

1. detectar termos no texto
2. alimentar cards explicativos
3. sustentar a traducao resumida das clausulas
4. gerar arquivos exportaveis de explicacao e rastreabilidade

## Funcao metodologica

No artigo, o dicionario pode ser apresentado como:

- artefato central da DSR
- base semantica reutilizavel do MVP
- mecanismo de explicabilidade do sistema
- ponte entre linguagem juridica, linguagem simples e representacao visual
