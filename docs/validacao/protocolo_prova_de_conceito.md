# Protocolo da Prova de Conceito

## Objetivo

Verificar se a apresentacao em linguagem simples com apoio visual melhora compreensao e leitura de clausulas em comparacao ao texto original.

## Desenho

- estudo comparativo de prova de conceito
- grupo A: leitura do texto original
- grupo B: leitura com o MVP

## Procedimento

1. apresentar termo de consentimento da pesquisa
2. aplicar questionario pre-teste
3. registrar tempo de leitura
4. aplicar teste de compreensao com 10 questoes
5. aplicar questionario pos-teste
6. consolidar resultados descritivos

## Medidas principais

- acerto no teste de compreensao
- tempo de leitura
- tempo de resposta
- percepcao de clareza
- percepcao de confianca
- utilidade percebida do MVP
