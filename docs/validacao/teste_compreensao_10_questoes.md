# Teste de Compreensao

1. A clausula informa quais dados sao coletados?
2. A clausula informa para que os dados serao usados?
3. A clausula menciona compartilhamento com terceiros?
4. A clausula informa se pode haver transferencia internacional?
5. A clausula menciona qual base legal sustenta o tratamento?
6. A clausula informa por quanto tempo os dados podem ser retidos?
7. A clausula apresenta algum direito do titular?
8. A clausula menciona canal para exercer direitos?
9. A clausula explica o que ocorre em caso de incidente de seguranca?
10. A clausula permite identificar um risco pratico ao usuario?

Observacao: para cada sessao, as opcoes devem ser adaptadas ao documento-base utilizado.
