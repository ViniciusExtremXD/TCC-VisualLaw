# System Usability Scale (SUS) — Adaptado

> Escala de usabilidade padrão ISO. Aplicar **após** o uso do MVP.
> 10 itens, Likert 1–5 (1 = Discordo totalmente, 5 = Concordo totalmente).

## Instruções ao Participante

"Para cada afirmação abaixo, marque o número que melhor reflete sua experiência com a ferramenta Visual Law."

## Cálculo do Score SUS

1. Para itens **ímpares** (1, 3, 5, 7, 9): Score = Resposta − 1
2. Para itens **pares** (2, 4, 6, 8, 10): Score = 5 − Resposta
3. Somar os 10 scores parciais
4. Multiplicar por 2.5
5. Resultado: **0–100** (média aceitável ≥ 68)

## Interpretação

| Score SUS | Classificação |
|-----------|---------------|
| 0–25      | Pior imaginável |
| 26–50     | Ruim |
| 51–67     | OK |
| 68–80     | Bom |
| 81–100    | Excelente |
